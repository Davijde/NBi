NBi is an open-source framework to test your Business Intelligence solutions.

Just plain xml

No need of C#, Java or any other development language to test your BI Solution. Your tests are written in xml by the means of a useful and intuitive syntax.

All your components

NBi helps you to create tests targeting your databases, cubes, etls and reports with a unique framework. You can natively connect to any database supporting OleDb or ODBC connections and by the means of extensions you can reach an additional bunch of NoSQL databases. When using the Microsoft BI stack, this framework let you test your SQL Server Database, SSIS, SSAS and SSRS components.

Automate generation

NBi is shipped with a tool to facilitate the automation of creation of test-cases. Build hundreds tests in a couple of minutes and facilitate the maintenance of these tests.

Compare two query results

<test name="Compare two query results">

  <system-under-test>

    <execution>

      <query>

        select * from FirstTable;

      </query>

    </execution>

  </system-under-test>

  <assert>

    <equalTo>

      <query>

        select * from SecondTable;

      </query>

    </equalTo>

  </assert>

</test>
